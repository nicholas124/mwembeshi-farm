## Design Philosophy

The Mwembeshi Farm app is designed with these principles:
- **Mobile-first**: Primary use case is phone in the field
- **Touch-friendly**: Large buttons, easy scrolling
- **Low-bandwidth**: Minimal data, offline capability
- **Intuitive**: Icons and colors over text where possible
- **Localized**: Ready for Bemba/Nyanja translation

## Color Palette

```
Primary Green:   #16a34a (Farm/nature theme)

Warning Yellow:  #f59e0b (Alerts)
Danger Red:      #dc2626 (Errors/delete)
Earth Brown:     #78716c (Earthy accent)
Background:      #f9fafb (Light) / #111827 (Dark)
```

## Typography

- **Headings**: Inter, semi-bold
- **Body**: Inter, regular
- **Numbers**: Tabular figures for data alignment

## Accessibility Guidelines

1. **Color Contrast**: Minimum 4.5:1 ratio
2. **Touch Targets**: Minimum 44x44px
3. **Font Size**: Minimum 16px base
4. **Icons**: Always with labels or tooltips
5. **Focus States**: Clear visible indicators
6. **Screen Reader**: Proper ARIA labels
